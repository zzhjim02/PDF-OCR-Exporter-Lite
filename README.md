# PDF OCR文本提取工具

一个图形化界面的工具，用于从PDF文件中提取OCR识别结果并保存为TXT文件。

## 功能特点

- 图形化界面操作，简单易用
- 支持批量处理大量PDF文件
- 保留原有的文件夹结构
- 支持双层PDF（包含OCR文本层）
- 多种提取方法可选
- 实时显示处理进度和日志


## 安装依赖

### 必需依赖

```bash
# 推荐使用 PyMuPDF（提取效果更好）
pip install pymupdf

# 或者使用 pdfplumber
pip install pdfplumber
```

### 可选依赖

两个库都安装可以获得更好的兼容性，程序会自动选择最佳方法。

## 使用方法

### 启动程序

```bash
python pdf_ocr_extractor.py
```

### 操作步骤

1. 点击"选择文件夹"按钮，选择包含PDF文件的源文件夹
2. 点击"选择文件夹"按钮，选择TXT文件的输出文件夹
3. 选择提取方法：
   - **自动选择**：程序自动选择可用的提取库
   - **PyMuPDF**：使用PyMuPDF库提取（推荐）
   - **pdfplumber**：使用pdfplumber库提取
4. 点击"开始提取"按钮开始处理
5. 处理完成后，可在输出文件夹查看提取的TXT文件

## 注意事项

- 程序会递归扫描源文件夹中的所有PDF文件
- 输出的TXT文件会保持与PDF文件相同的文件夹结构
- 如果PDF文件没有OCR文本层，提取结果可能为空
- 对于扫描版PDF（图片），需要先进行OCR识别才能提取文本

## 关于双层PDF

双层PDF是指包含两层内容的PDF文件：
1. 图像层：原始扫描图像
2. 文本层：OCR识别结果（透明文本）

本工具专门用于提取双层PDF中的OCR文本层内容。

## 系统要求

- Python 3.6 或更高版本
- Windows / macOS / Linux
